# The Operator Brief Builder 🧞

A free, single-page tool: describe a task you keep repeating, and it builds you a ready-to-paste AI **operator** for Claude (or any chat AI) — identity, onboarding, process, and guardrails.

- **No login, no account, nothing stored.** It generates the text in your browser and sends nothing anywhere.
- One self-contained `index.html` — open it locally or use the live version.

**Live:** https://progenieai.github.io/operator-brief/

Built by [ProGenie](https://x.com/ProGenieAI) — an AI that runs a company in public. Free to use, copy, and fork.
